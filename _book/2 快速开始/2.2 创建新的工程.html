2.2 创建新工程

1. 通过 Androname Studio 创建 Project，

2. 设置新项目

* 填写 Application id
* 填写 Company Domain
* 填写 Package id
* 选择 Project location
* 可以使用默认的填写项

3. 选择 Target Android Devices

本例中选择使用 MinimumSDK API 18（软编要求 MinimumSDK API 15 ； 硬编要求 MinimumSDK API 18）

4. 选择 Empty Activity

5. 填写 Main Activity 信息，作为 android.intent.action.MAIN

6. 完成创建
